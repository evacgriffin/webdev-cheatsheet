HTML

HTML (Hyptertext Markup Language) is the markup language responsible for displaying webpages in web browsers. HTML determines the general layout of a website. With HTML, web developers can add components such as text, images, videos, audio, and links to their webpages.

Tag (Attributes) Description
GENERAL
<!DOCTYPE html> document type declaration, start of every html document
<html>, </html> (lang) beginning and end of the html document (document language)
<body>, </body> beginning and end of visible part
TEXT LAYOUT
<h1>, </h1> ... <h6>, </h6> headings, h1 - most important, h6 - least important
<p>, </p> defines a paragraph
<br> line break
<b>, </b> or <strong>, </strong> defines bold text
<i>, </i> or <em>, </em> defines italic text
<sub>, </sub> or <sup>, </sup> defines subscripted/superscripted text
MEDIA
<a>, </a> (href) defines a link (link destination)
<img> (src, alt, width, height) defines an image (source link, alternate description, image width and height)
LISTS
<ul>, </ul> unordered list
<ol>, </ol> ordered list
<li>, </li> denotes each list item
TABLES
<table>, </table> defines an html table
<tr>, </tr> defines a row of the table
<th>, </th> defines a table header
<td>, </td> defines table data for one cell of the table

CSS

CSS (Cascading Style Sheets) is used to determine the physical appearance and style of a webpage. CSS is used in combination with HTML to give elements of a webpage different styles. For example, color, fonts, sizing, and spacing.

Property Description
COLOR
color: value in hex, rgb, or standard color name sets the text color
background-color: value in hex, rgb, or standard color name sets the background color of an html element
opacity: value between 0.0 - 1.0 specifies transparency of a color, 0.0 is most transparent, 1.0 is least transparent
PAGE LAYOUT
border: width style color sets border width, border style (solid, dotted, etc.), and border color
border-radius adds rounded corners to a border, 50% creates circular border
padding: top right bottom left creates space around content but inside the border
box-sizing: border-box keep defined width no matter the amount of padding
margin: top right bottom left creates space around an element outside of the border
margin: auto centers the element horizontally in its container
TEXT
text-align: left, right, center, or justified aligns text according to the specified value
text-decoration: line color style thickness adds text decoration, such as underline or line-through
text-transform used to change capitalization
letter-spacing used to change spacing between characters
word-spacing used to change spacing between words
FONTS
font-family specifies the font family
font-style sets text style to normal, italic, or oblique(leaning)
font-weight sets the font weight
font-size specifies the font size
