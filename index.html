HTML

HTML (Hyptertext Markup Language) is the markup language responsible for displaying webpages in web browsers. HTML determines the general layout of a website. With HTML, web developers can add components such as text, images, videos, audio, and links to their webpages.

Tag Description
<!DOCTYPE html> document type declaration, start of every html document
<html>, </html> beginning and end of the html document
<body>, </body> beginning and end of visible part
<h1>, </h1> ... <h6>, </h6> headings, h1 - most important, h6 - least important

CSS

CSS (Cascading Style Sheets) is used to determine the physical appearance and style of a webpage. CSS is used in combination with HTML to give elements of a webpage different styles. For example, color, fonts, sizing, and spacing.
